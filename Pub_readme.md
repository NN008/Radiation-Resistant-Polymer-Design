Title: Closed-Loop Generative Design for Radiation-Resistant Polymers
Description:
This repository contains the supporting data and pseudocode for the manuscript:
A Closed-Loop Deep Generative Model for the Inverse Design of Radiation-Resistant Polymers.
Contents:
PI1M_Tg_MAC.csv — Augmented PI1M dataset containing standardized SMILES, 17 RDKit descriptors, and filled-in glass transition temperature (Tg) and mass attenuation coefficient (MAC) values. Experimental values are included where available; surrogate predictions were used to fill gaps.
PolymerDesign_Pseudocode.pdf — High-level pseudocode (non-executable) describing the closed-loop workflow used in this study, including dataset preparation, surrogate modeling, polymer generation, scoring, and diversity-based selection.
Notes:
The pseudocode is provided for transparency and reproducibility.
Full implementation code is withheld due to ongoing development and potential intellectual property considerations.
The dataset is sufficient to replicate the surrogate modeling approach and descriptor pipeline described in the paper.
